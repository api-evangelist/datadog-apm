The Pup CLI enables AI agents to securely access Datadog platform data from the command line without requiring long-lived API keys, reducing credential risk in agentic workflows. Developers can grant AI agents scoped, time-limited access to metrics, logs, traces, and dashboards to power automated investigation and remediation tasks.
